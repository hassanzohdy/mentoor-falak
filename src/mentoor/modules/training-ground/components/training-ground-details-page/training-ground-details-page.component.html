### Please consider the following notes when pushing to Github. - Make sure you organize your files in its proper folder, i.e don't put different types in folder. - Always commit the changes you made in the file and write a proper commit message based on your change, i.e - DO not use `git add .` but use `git add my-file.html` instead. - Always write a `Change Log` at the end of the readme.md file to define what updates have you done in the repository.
