Topic suggestion

You can suggest a topic to be included in the track topics list.

Once the form is submitted, it will be previewed and checked. If the suggestion is approved, you'll receive

Notes before writing.

  • Make sure the topic is not listed in the topics list.
  • The topic MUST BE related to the track objectives only.
  • The topic MUST BE the same level as the track, for example if the track is for beginners DO NOT add an advanced topic..
